Детекция сотрудников на видео с  магазинов.
Решение:

Использован ансамбль из 5 моделей YOLO разных архитектур: YOLOv8n, YOLOv8s, YOLOv9c, YOLO11n, YOLO26n

Применена 5-кратная стратифицированная кросс-валидация (стратификация по наличию класса staff)

Кастомная аугментация через Albumentations: Blur, MedianBlur, ToGray, CLAHE

Оптимизация порогов уверенности для каждой модели отдельно (максимизация F1-меры на валидации)

Weighted Boxes Fusion (WBF) для ансамблирования предсказаний с весами, пропорциональными mAP50 каждой модели

AdamW оптимизатор + косинусное расписание скорости обучения

Early stopping (patience=5) для предотвращения переобучения

Размер входного изображения: 800×800 пикселей

Результат: Итоговый файл submission_final.csv с детекциями в формате JSON (координаты центров, ширина, высота, confidence) для 4454 тестовых изображений. Метрика качества на валидации — mAP50 ~0.93,
