В рамках данного проекта я обучал несколько разных моделей для классификации 15 классов фруктов и овощей на прилавках магазина.
Решение:

Использована предобученная EfficientNet-B3 (ImageNet) с дообучением на целевых данных

Применена 5-кратная кросс-валидация для повышения устойчивости модели

Реализованы аугментации (RandomResizedCrop, горизонтальные отражения, повороты, ColorJitter)

Использован EMA (Exponential Moving Average) для сглаживания весов модели

Label smoothing (0.05) для регуляризации и снижения переобучения

Оптимизатор AdamW + CosineAnnealingLR для управления скоростью обучения

TTA  — усреднение предсказаний по 4 вариантам (оригинал/отражение + два размера)

Ансамбль 5 моделей — усреднение вероятностей всех фолдов
